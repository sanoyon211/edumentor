## 📖 Project Overview

**The EduMentor** is an educational platform designed to bridge the gap between students and tutors. Users can explore available tutors, browse subjects and categories, and find the right mentor for their learning journey. The platform features a clean, intuitive UI for a smooth educational experience.

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React.js | Component-Based UI |
| JavaScript (ES6+) | Logic & Functionality |
| Tailwind CSS | Responsive Styling |
| CSS3 | Custom Design Elements |

---

## ✨ Key Features

- 🧑‍🏫 Browse and explore tutors by subject/category
- 📱 Fully **responsive** layout for all devices
- 🔍 Search and filter functionality
- 🎨 Clean and modern educational UI
- ⚡ Fast SPA experience with React Router

## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/sanoyon211/edumentor.git

# 2. Navigate into the project folder
cd edumentor

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
# OR
npm start

# 5. Open in browser
# Visit: http://localhost:5173 or http://localhost:3000
```
